
我们的目的是为web开发者提供一个公共的静态资源库，让它尽可能的收录开源库，并为之提供免费的CDN加速服务，使之更加稳定、高速，以方便广大开发者使用。

您可以在我们的github提交开源库(JS、JQuery、css、image等静态文件)，我们审核通过后，你可以在自己的网页上直接通过script标记引用这些资源，这样做不仅可以为您节省流量，还能通过CDN加速，获得更快的访问速度。
